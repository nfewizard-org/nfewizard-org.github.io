    Inicializar Ambiente

    Repositório do projeto NFeWizard-io.

    ATENÇÃO: Após a inicialização do ambiente você será capaz de utilizar a biblioteca para se comunicar com os webservices da SEFAZ. É importante que os  objetos (dados) fornecidos aos métodos da biblioteca estejam na mesma ordem dos exemplos, uma vez que os XML serão gerados na mesma ordem e  validados pelo webservice.


    Para mais informações sobre opções de configuração consulte a sessão Configuração de Ambiente.

    Instalação

    npm install nfewizard-io
    yarn add nfewizard-io
